Silverhound

iOS app giving daily silver values for many common coins.

View on GitHub Download .zip Download .tar.gz

Silverhound is a simple but full-featured app that estimates the silver value of common but often overlooked coins.

The app is a heavy user of iOS-native capabilities, especially Core Data. Core Data is pretty magical once you get things wired up.
